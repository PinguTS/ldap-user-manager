## Screenshot Guidelines

### Technical Requirements
- **Resolution**: 1920x1080 or higher
- **Format**: PNG or JPEG
- **Quality**: High quality, clear text
- **Browser**: Use consistent browser (Chrome/Firefox)

### Content Guidelines
- **Use real data**: Show realistic examples, not placeholder text
- **Highlight features**: Use annotations to point out key features
- **Consistent styling**: Use same theme/colors across all screenshots
- **Accessibility**: Ensure good contrast and readable text

### File Naming
- Use descriptive, lowercase names with hyphens
- Include feature name in filename
- Example: `user-management-dashboard.png`
